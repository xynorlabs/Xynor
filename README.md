# Xynor Labs — AI Engineering Company

Production-ready website built with Next.js 15, React 19, Three.js, and Framer Motion.

## Setup

```bash
npm install
npx prisma migrate dev
npm run dev
```

## Stack

- **Framework**: Next.js 15 (App Router)
- **UI**: React 19, Tailwind CSS, ShadCN
- **3D**: Three.js, React Three Fiber
- **Animation**: Framer Motion, GSAP
- **State**: Zustand
- **DB**: PostgreSQL + Prisma
- **Auth**: Auth.js
- **Storage**: Cloudinary

## Architecture

```
app/         # Routes & API
components/  # Reusable UI
lib/         # Utilities & config
actions/     # Server actions
api/         # API routes
prisma/      # Database schema
store/       # Global state
hooks/       # Custom hooks
types/       # TypeScript types
styles/      # Global styles
```

## Features

- Interactive 3D AI Core (Three.js)
- Glassmorphism UI system
- Smooth scroll (Lenis)
- AI Playground (interactive demo)
- Full Prisma schema
- Server actions + API routes
- Responsive (mobile-first)
- SEO optimized
```
